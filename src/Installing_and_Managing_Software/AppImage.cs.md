# AppImage

![AppImage|100x100, 100%](../img/AppImage.png)

AppImage je univerzální formát balíčku, který se pokouší spojit závislosti, které aplikace vyžaduje, do jednoho přenosného souboru. Lze je nainstalovat stažením libovolného souboru s příponou `.AppImage` a následným udělením oprávnění ke spuštění ve vlastnostech souboru, aby aplikace mohla správně běžet. Stejně jako spustitelné soubory systému Windows se obvykle nacházejí na webových stránkách projektu ke stažení. AppImages byste měli stahovat pouze ze zdrojů, kterým důvěřujete.

## Řadicí páka

![Řadicí páka|500x500](../img/Gear_Level.png)

Aplikace nainstalované jako AppImage se mohou integrovat s vaší plochou a být spravovány pomocí [Gear Lever](https://github.com/mijorus/gearlever), kterou si můžete nainstalovat v [Bazaar](./Flatpak.md).  Gear Lever také zkontroluje nové aktualizace, ale **upgrady bude i tak nutné použít ručně**. Pro správu AppImages důrazně doporučujeme nainstalovat Gear Lever.

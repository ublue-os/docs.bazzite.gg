# Introduction au jeu sur Bazzite

## Changement de paradigme

Le fait que les jeux sous Linux constituent désormais une alternative viable à Windows est un phénomène relativement récent. La plupart des jeux exclusifs à Windows peuvent fonctionner sur Bazzite avec une configuration moderne. Il reste certes quelques imperfections, mais le jeu sous Linux offre des avantages une configuration adaptée, comme des temps de rafraîchissement supérieurs à ceux de Windows.

### Jeux Steam

Steam est la solution recommandée pour jouer à des jeux PC sur Bazzite, car il intègre une couche de compatibilité appelée [**Proton**](https://steamcommunity.com/games/221410/announcements/detail/1696055855739350561),  qui est un fork de [**WINE**](https://www.winehq.org/) et de divers autres projets fusionnés en un lanceur pratique intégré à Steam pour Linux, destiné aux jeux développés par Valve. La plupart des jeux PC solo devraient pouvoir fonctionner sans aucun réglage particulier, mais il arrive parfois qu'un jeu acheté et installé via Steam ne fonctionne pas dès sa première utilisation.  Certains jeux multijoueurs en ligne plus récents nécessitent un accès plus approfondi au matériel de votre ordinateur et aux pilotes Windows pour faire fonctionner leur logiciel anti-triche, qui peut ne pas fonctionner sur Bazzite, ces jeux seront injouables à moins que Windows ne soit en cours d'exécution.

### Jeux hors Steam

Bazzite permet également d'exécuter des jeux hors Steam à l'aide de **Lutris**, une interface permettant de lancer des jeux et d'autres logiciels via Proton/WINE. [**Faugus Launcher**](https://flathub.org/fr/apps/io.github.Faugus.faugus-launcher) est une alternative à Lutris disponible dans la boutique d'applications Bazaar, possèdant une conception plus simple que celle de Lutris. [**Heroic Games Launcher**](https://flathub.org/fr/apps/com.heroicgameslauncher.hgl) est recommandé plutôt que Lutris pour les jeux achetés sur Epic Games Launcher, GOG et Amazon Games Launcher.  

Les jeux installés depuis le Microsoft Store ne fonctionnent **pas**, sauf si vous utilisez un client Xbox Cloud Gaming tel que [**Greenlight**](https://github.com/unknownskl/greenlight), qui nécessite un abonnement Game Pass Ultimate pour la plupart des jeux. Fortnite peut également être joué via cette méthode **sans** abonnement Game Pass.  Il existe également [**certains titres Game Pass disponibles sur Battle.net**](https://us.support.blizzard.com/en/help/article/000367382) qui peuvent être lancés à l'aide de Lutris.  

Les émulateurs de console peuvent être installés depuis Bazaar ou via des projets tels que [**Retrodeck**](https://flathub.org/en/apps/net.retrodeck.retrodeck) ou [**Emudeck**](https://www.emudeck.com/) (disponibles dans l'application Portail Bazzite).
